Project Overview
This is a fully functional Claim Management & Compensation System (CMCS) built with ASP.NET Core MVC (.NET 8) for lecturers, programme coordinators, and academic managers.

Features Implemented (100% Complete)
Lecturer can submit monthly claims (hours worked + supporting documents)
Claims appear instantly in My Claims page
Programme Coordinator can Verify (approve/reject) pending claims
Academic Manager can Approve/Reject verified claims
File upload system (PDF, Word, etc.) with safe storage in wwwroot/uploads
Real-time total amount calculator on submit page
Responsive Bootstrap UI
Full role-based authorization (Lecturer / ProgrammeCoordinator / AcademicManager)
SQLite database with proper relationships
